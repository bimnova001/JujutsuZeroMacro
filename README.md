# Jujutsu Zero Raid Macro

**Version:** 1.1

This is an **AutoHotkey Macro** for the game **Jujutsu Zero (Raid)**.  
It automates actions for the **Toji Raid** boss fight.

---

## Key Features

- Press **N** to **lock-on the target**  
- Smooth camera rotation for proper positioning  
- Automatic skill usage: **Gojo Skill C** and **Sukuna Skill C**
- Click **Raid** positions and **Retry** automatically
- ImageSearch check is game start ?

---

## Configuration

- **Setting before start**  
  - Set lock on : N

---
## Installation

1.1 Download and install [AutoHotkey](https://www.autohotkey.com/)  
1.2 Place the `.ahk` file in your desired folder  
1.3 Double-click the `.ahk` file to run the macro 
Or 
2.1 Downlaod .exe
2.2 Run
---

## Usage

1. Enter **Jujutsu Zero** and go to the **Toji Raid**  
2. Press **F1** to start the macro  
3. Wait for the macro to complete its actions  
4. Press **F2** to stop the macro  
5. Press **Esc** to exit the program
